# Better Image Upload

A Revenge plugin for Discord that removes the 10 image upload restriction.

## Features

- Upload more than 10 images at once
- Automatically splits uploads into batches of 10
- Shows progress notifications for each batch
- Preserves your message content on the last batch

## Installation

Install via Revenge plugin manager using the repository URL.

## Usage

Simply select more than 10 images when uploading to Discord. The plugin will automatically split them into batches and upload them sequentially.
